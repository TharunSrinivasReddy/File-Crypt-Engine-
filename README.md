# ImageTextEncryptor

A Python-based encryption and decryption tool for both text and images using:

- **Text Encryption**: `cryptography.fernet`
- **Image Encryption**: `AES` (from `pycryptodome`)
- **GUI**: `Tkinter` for user-friendly interface
